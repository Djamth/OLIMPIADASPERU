#  Sistema de Olimpiadas – Arquitectura SOA

Sistema de gestión de olimpiadas deportivas basado en Arquitectura Orientada a Servicios (SOA), diseñado para ser escalable, modular e interoperable.

---

## 📌 Características

- 🔹 Arquitectura SOA  
- 🔹 Servicios desacoplados  
- 🔹 Escalabilidad horizontal  
- 🔹 Integración mediante eventos  
- 🔹 Despliegue con Docker

## ⚙️ Instalación

```bash
# Clonar repositorio
git clone https://github.com/djamth/OLIMPIADASPERU.git

# Backend
cd backend
./mvnw spring-boot:run

# Frontend
cd frontend
npm install
ng serve
